# Neuro-Semantic Suicide Risk Detection Framework

This repository contains the official implementation of the paper:

**"An Explainable Multimodal Framework for Suicide Risk Detection using an Electroencephalogram and Text Dataset"**

## Requirements

- Python 3.10+
- PyTorch 2.1+
- Transformers 4.30+
- MNE-Python 1.5+
- Captum 0.6.0+

Install dependencies:
```bash
pip install -r requirements.txt

Data Availability Statement

All datasets used in this study are publicly available and were accessed in accordance with their respective license terms. The Temple University Hospital EEG Corpus (TUH EEG) is available at https://www.isip.piconepress.com/projects/tuh_eeg/ under a data use agreement. PhysioNet Mental Imagery EEG (PhysioNet MI) is accessible via https://physionet.org/content/eegmmidb/1.0.0/ under the Open Data Commons Attribution License. The Reddit Suicidality Dataset (RSI) and CLPsych Shared Task corpora are available through their respective repositories: RSI at https://github.com/ji/suicide-ideation-detection and CLPsych at https://clpsych.org/. The DAIC-WOZ corpus is accessible via the University of Southern California’s Institute for Creative Technologies (https://dcapswoz.ict.usc.edu/).
