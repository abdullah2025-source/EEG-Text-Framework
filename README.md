# Neuro-Semantic Suicide Risk Detection Framework

This repository contains the official implementation of the paper:

**"An Explainable Multimodal Framework for Suicide Risk Detection using an Electroencephalogram and Text Dataset"**

## Requirements

- Python 3.10+
- PyTorch 2.1+
- Transformers 4.30+
- MNE-Python 1.5+
- Captum 0.6.0+

Install dependencies:
```bash
pip install -r requirements.txt
